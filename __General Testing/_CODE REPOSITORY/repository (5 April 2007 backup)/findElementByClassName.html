Find an Element Sequentially By Class Name

Why would you need to do this?
·One of our clients used different tags for dynamic text, but the same class name to identify them. we need to pull them out sequentially cuz we will be using the same tag on our end to write it out in our template. So we can't pull them out by tag name, UNLESS we use '*' as the tag.

In this example, we use document.getElementsByTagName('*'), which is the W3C version of document.all

to change all the tags with class='listurl', and see what they are.
