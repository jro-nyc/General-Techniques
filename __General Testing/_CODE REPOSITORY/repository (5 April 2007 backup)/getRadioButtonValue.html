Get the Radio Button or CheckboxValue - How do we get them?

Getting the value of text boxes, selects, text areas, etc is easy. Just read the value directly. But checkboxes and radio buttons are a little more work.

You actually need to iterate through each value and see if the button is checked. Then if so, save that button's value as the answer to the question.

For a checkbox, there is only one button, so you iterate through its one value.
