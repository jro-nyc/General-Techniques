Remove a HTML object from the DOM using the TAG name (document.getElementsByTagName).

Why would you need to do this? Well, this is useful when we are reading dynamic content via the DOM and are re-rendering it in our code in some manner, and we would like to reuse the same name or IDs. And you can cover more general things than having to know each individual ID. So say you wanted to read a bunch of hidden inputs and copy them over and remove the orginals from the DOM? Use TAG=input, and type=hidden, done. And thus it's very useful for form pages because the you can use their client-side correction code
