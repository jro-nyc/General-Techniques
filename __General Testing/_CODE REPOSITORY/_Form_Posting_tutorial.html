Form POST tuturial - What happens to a button/image input when it is clicked to submit a form?

I created this file to show that we can differentiate which image or button a user clicked on when submitting a form.

What's the point? Well, I want all of us to understand what's going on with forms as well as possible. And on some forms, there may be multiple mini "forms" that can be differentiated by which button is clicked.

Pick either one of the two "mini forms" to submit, or one of the buttons at the bottom, make sure you have "debug clickthrus" on, and see what gets posted.
