Focus on First Form Element

This page has 3 forms with the same stuff, but the order of the inputs is changed. When the Page comes up, focus is on the first element of the last form. You can then click on the buttons at bottom to focus on whichever form you'd like.

View the source and see how it's done. Quite simple, just iterate through a form object by elements - ie sequential order - and put focus on the first non-hidden input.
