Remove a HTML object from the DOM using the ID (document.getElementById).

Why would you need to do this? Well, this is useful when we are reading dynamic content via the DOM and are re-rendering it in our code in some manner, and we would like to reuse the same name or IDs. Very useful for form pages because the you can use their client-side correction code
