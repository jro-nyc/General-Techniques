Fiddler Setup

Fiddler  is an HTTP debugging proxy server application. Though capable of many tasks, Optimost primarily uses Fiddler to manipulate outgoing file requests and incoming content data. 

Download Link

Fiddler4, can be downloaded here .

Once fiddler is downloaded, there are a few things we need to setup to make sure it functions properly.

Configure Fiddler to Decrypt HTTPS Traffic

As more sites move to HTTPS, we need to allow Fiddler to handle that traffic. Follow these steps to prevent all connections from looking like this:

  1. Click Tools > Telerik Fiddler Options > HTTPS
  2. Click the HTTPS tab, and click Decrypt HTTPS Traffic box.
  3. Click the OK button.

Required Extensions

There are a few extensions dedicated to making Fiddler easier to use, and they provide an easy method of accomplishing common required tasks. The three extensions we require are: SyntaxView Inspector, RulesTab2, and FiddlerScript Editor. Luckily, these extensions are bundled together in Syntax-Highlighting Add-ons package found here . Installation is straightforward. Once installed, restart Fiddler.
