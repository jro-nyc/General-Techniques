Introduction

Working on an Aussie site - same area as the previous entry - Making mileage in auto advert BOLD - one of the TCs needed to take "car synopsis" text and make it bulleted text. This means using commas to find the separation points for the bullets, except for the mileage, where the commas should be ignored for bulleting purposes. Here we go.

Creating the regexp value

So I am looking to use commas to separate the bullet areas. But I need to not do that for the mileage, so there I am looking for numbers, comma-separated. Similar to the previous example. For that, the code looks like: 

Here is the HTML:
<span itemprop="description"> Buttercup Yellow, 2D COUPE, 64,000 kms, 6 SP AUTO DIRECTSHIFT, 6 Cylinder</span>
<br/>
var spans=document.getElementsByTagName("span");
num=0;
for(var i=0,len=spans.length;i<len;i++){
    if(spans[i].innerHTML.indexOf("kms")!=-1){
        var myVal=spans[i].innerHTML;
        var regExVal=/\d{1,3},\d{1,3} kms/;
        if(myVal.match(regExVal))myVal=myVal.replace(regExVal,myVal.match(regExVal).toString().replace(/,/,"."));
        bullets=myVal.split(",");
        list='<ul>';
        for(var j=0,len2=bullets.length;j<len2;j++){
               var regExVal2=/\d{1,3}.\d{1,3} kms/;
               if(bullets[j].match(regExVal2))bullets[j]=bullets[j].replace(/\./,",");
               list+='<li>'+bullets[j]+'</li>'; 
        }
        list+='</ul>';
        spans[i].innerHTML=list;
    }
}

The variable regExVal contains our string - /\d{1,3},\d{1,3} kms/ - which can be broken down into:

  1. The '/' before and after demark my regexp value, like a quote around a string.
  2. The '\d' means I am looking for numbers.
  3. The '{1,3}' means there will be 1 to 3 digits.
  4. The ',' means I am looking for a domma.
  5. The '{1,3}' means there will be 1 to 3 digits, again.
  6. The ' kms' means i am looking for ' kms'
