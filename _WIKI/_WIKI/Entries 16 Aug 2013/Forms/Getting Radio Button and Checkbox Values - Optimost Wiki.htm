Getting Radio Button and Checkbox Values

From Optimost Wiki

Jump to: navigation, search

From Jim: Getting the value of text boxes, selects, text areas, etc is easy. Just read the value directly. But checkboxes and radio buttons are a little more work.

You actually need to iterate through each value and see if the button is checked. Then if so, save that button's value as the answer to the question.

For a checkbox, there is only one button, so you iterate through its one value.

Getting Radio Button Values

function getRadioValue(){
	btn=document.forms['form1'].OPTIN_CONST;
	val='';
	for(var i=0;i<btn.length;i++){
		valid = btn[i].checked;
		if (valid) {
                  val=btn[i].value;
                  break;
                }
	}
	//alert(val!=''?'The value for the radio button is '+val+'':'If don\'t check on of the radio buttons, there is no value');
        return val;
}



Getting Checkbox Values

function getCheckboxValue(){
	btn=document.forms['form1'].checkIt;
	val='';
	valid = btn.checked;
	if (valid) val=btn.value;
	//alert(val!=''?'The value for the checkbox is '+val+'':'If you don\'t check the box, there is no value');
        else val="Not Checked";
        return val;
}

If you wanted a list of values that are checked in a list of checkboxes, you could add the values to an array instead and return that.
