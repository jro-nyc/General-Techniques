Cookies Overview

Various features require the use of cookies. Most of these features allow you to customize cookie name and domain.

Cookie Name

Each cookie is given a name that the browser uses to reference the cookie. Each name should be unique. Names can only consist of alphabetic characters A-Z, hyphen (-) and underscore (_) and cannot be longer than 64 characters in length.

Cookie Domain

Each cookie is associated with a domain that controls where the cookie is valid. Domains other than the one specified cannot access the cookie value. You have three options for cookie domain.

  • The domain in the address bar
    Cookies with this option will be valid for the domain shown in the browser address bar. For example, if the address bar shows the domain www.acme.com the cookie will be valid for acme.com and all subdomains of acme.com. If you do not want the domain to be accessible to subdomains click Restrict to complete domain and the cookie will be limited to the domain as shown in the address bar.
  • The domain:
    Cookies with this option will be valid for the domain specified by the domain name field. Use this option instead of The domain in the address bar if the domain shown in the address bar is different from the domain where the cookie is referenced. Check Include subdomains if you want the cookie to be valid for subdomains of the specified domain.
  • Cross domain
    Use this option when the cookie is initially set from one domain and is referenced by a different domain. For example, if a cookie is set by the domain www.acme.com but is referenced by the domain www.acmestore.com the cookie must be cross domain. Using cross domain cookies is not recommended because they are more likely to be rejected by browsers.


The first two options set first-party cookies. Browsers will not block first-party cookies except under the most strict security settings. The cross domain option sets a third-party cookie. Browsers are more restrictive with third-party cookies and may block them under normal security settings. For this reason, first-party cookies are recommended.


The domain used for cross domain cookies is .optimost.com which allows cross domain cookies to be referenced by all trial, click thru and counter requests.
