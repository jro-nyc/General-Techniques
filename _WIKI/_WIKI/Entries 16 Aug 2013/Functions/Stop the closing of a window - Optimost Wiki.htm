Stop the closing of a window

From Optimost Wiki

Jump to: navigation, search

Preventing visitor exit temporarily

You can do this by adding an window.onbeforeunload event that does something to try to stop you from leaving.

Code

I setup a simple example based on Google's home page, so:

  1. Go to google.com
  2. Cut and paste this code in the Console tab (using the appropriate plugin/option in IE, FF, Chrome, etc)
  3. Run the code
  4. Close the window/tab google.com is in.


Three things to know about this code

  1. You create a window.onbeforeunload event and it needs to contain:
    • A function that adds something visible to the page that somehow tries to convince the visitor not to leave.
    • A function that turns OFF this window.onbeforeunload event so that this does not repeat again and really annoy the visitor
    • Set the return value to "return false" to "undo" the closing of the window.
  2. Having attempted to close the window, now the visitor is presented with an alert from the browser asking them if they want to leave this page or stay on it.
  3. At the same time, the function we added will display something that is trying to convince the visitor to stay. This will usually be
    • Some layover - usually called a "lightbox" - that gives you more options for staying. This is what I have added, although mine is way simpler than anything you'd consider doing.
    • A alert asking a visitor not to leave. This was done more in the past, you won't see this used as much today.
  4. Present with this information, you select to leave or stay.

Code

Here is the code. 

$opt={};
window.onbeforeunload = function(){
        $opt.popUp();
        $opt.eventStop()
        return false;
}
$opt.popUp=function(){
    var xyz=document.createElement("div");
    xyz.id="please";
    xyz.style.cssText="position:absolute;top:100px;left:20px;background-color:#FFFFcc;";
    xyz.innerHTML='My name is Jaime<br/>'+
                'please don\'t go....<br/>please don\'t go....<br/>please don\'t go....<br/>';
    document.body.insertBefore(xyz,document.body.firstChild);
    $opt.count=0;
    $opt.bkgColor=["#CCCCCC","#CC6600","#33CC00","#FFFFCC"];
    $opt.flickerPlease=function(){
       var obj=document.getElementById("please");
       if(obj)obj.style.backgroundColor=$opt.bkgColor[$opt.count];
       $opt.count++;
       if($opt.count>3)$opt.count=0;    
    }
    window.setInterval($opt.flickerPlease,1000);

}
$opt.eventStop=function(){
    	window.onbeforeunload = function(){};
}
