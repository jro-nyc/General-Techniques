Capitalizing First Letter Of Each Word (but not the Articles)

Suki was working on a template where the customer wanted dynamic content in a properly capitalized headline to have a value where the first letter of each word was capitalized, provided the word was not an article. So we worked together to create this function so everyone else could use it if they are ever asked to implement it in one of your templates.

The Code

Here is the code. There are two areas that start with identical code, and when the page loads, the bottom area - which is inside a DIV with id="newExample" - we take that area, find the h1 that either contains a or a text node, and capitalizes the first letter of each word.

Function: convertToFirstLetterCapsIfNotAnArticle

This is passed a string, and it goes through the string, breaks them into words. A word is discovered when we fine a space, or the end of the string. Then we go through each word, and look to see if it matches our list of articles, and if it does, it is not capitalized. If it does not match an article, we do capitalize it, using toUpperCase() on the first character of the word.

Function: capitalizeFirstLetterOfAllWords

Takes an ID passed to it and loops through to look for all the H1 tags to capitalize. If an object with the ID does not exist, just use the document object as our starting point. We then find the H1s, then if they also contain a SPAN tag, go inside that node, too. SO either from an H1 or its inside SPAN, we then get the firstChild, pull out its nodeValue, then that string to convertToFirstLetterCaps(str), which returns a string with the first letters capitalized.

Note that I also change the font-size of either the H1 of the SPAN inside and H1 to 24px. I did this by writing out a style, which is easier than doing DOM manipuation of the style. There is one big disadvantages doing it this way.

  1. You can only do a document.write of a style before the page is loaded. A document.write after a page load blanks out the page.

So if you need to do this after page load - ie an AJAX page needs this - you will have to change the style via the DOM.

<head>
<style type="text/css">
body{font-family:arial,sans-serif} 
TD.txt {font-family:verdana,sans-serif;font-size:small;}
h2{font-size:12px}
h1{font-size:32px}
</style>
<SCRIPT LANGUAGE="JavaScript">
function convertToFirstLetterCapsIfNotAnArticle(str){
	words= new Array;
	index=0;
	newStr=str;
	last=0;
	for(var i=1;i<str.length;i++){
		if(str.substring(i,i+1)==' '){
			words[index]=str.substring(last,i+1);
			last=i+1;
			index++;
		}	
	}
	if(last!=str.length){
		words[index]=str.substring(last,str.length);
		index++;
	}
	str2='';
	articles= new Array('a','on','the','with','of','an','and','in','to');
	for(var k=0;k<index;k++){
		arts=false;
		len=words[k].length;
		for(var j=0;j<articles.length;j++){
			if(words[k].substring(0,words[k].length-1)==articles[j])arts=true;
		}
		if(!arts)words[k]=words[k].substring(0,1).toUpperCase()+words[k].substring(1,words[k].length);
		str2+=words[k];
	}
	return str2;
}


function capitalizeFirstLetterOfAllWords(objId){
	obj=document.getElementById('newExample');
	if(obj==null) {
		obj=document;
		objId='';
	}
	c_h1 = obj.getElementsByTagName("h1");
	if (c_h1.length>0) {
		for(k=0;k<c_h1.length;k++){
			c_h1_span=c_h1[k].getElementsByTagName('span');
			if(c_h1_span.length>0){
				chds=c_h1_span[0].childNodes;
				//alert('There are '+chds.length+' child nodes');
				if(chds.length>0){
					str=c_h1_span[0].firstChild.nodeValue;
					str2=convertToFirstLetterCaps(str);
					c_h1_span[0].firstChild.nodeValue=str2;
					document.write('<style>#'+objId+' h1 span{font-size:24px}<\/style>');
				}		
			}
			else{			
				chds=c_h1[k].childNodes;
				//alert('There are '+chds.length+' child nodes');
				if(chds.length>0){
					str=c_h1[k].firstChild.nodeValue;
					str2=convertToFirstLetterCaps(str);
					c_h1[k].firstChild.nodeValue=str2;
					document.write('<style>#'+objId+' h1{font-size:24px}<\/style>');		
				}
			}	
		}
	}
}
