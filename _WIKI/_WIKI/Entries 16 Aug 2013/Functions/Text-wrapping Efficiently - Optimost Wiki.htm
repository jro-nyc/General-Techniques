Efficient use of space in areas containing text

We are doing an IDOL web implementation where we get products to display on an online retailer's web page. The retailer want us to put a product's details into two lines that each have enough room for about 20 characters or so, and if the details are not finished by the end of the second line, we add "..." at the end of the line.

The issue is that HTML will wrap text to the next line automatically based on the length of the last word on a line. So two 40 character strings may be formatted very differently based on where in the text the longest words exist.


For example, below are two string, both contain 65 characters. If you had a space that was about 25 characters wide, how many lines will they take to fit in that area?

  • "Our latest: East Land Bay Pinot Noir is just the taste to go with" - Fits in 3 lines comfortably, all about the same length
  • "Our latest: Parminversartiriani Rose is just the taste to go with" - Fits in 4 lines because of the long word "Parminversartiriani" which cannot fit on the first line, leaving us with a short 1st and 4th line

The solution

The solution is to allow us to cut words in the middle and wrap them to the next line. For a practical application with these kinds of details, the customer was happy to show it in this form without any punctuation since it allowed them to maximize the detail available to their customers at a glance.

The Code

This code accepts three parameters:

  1. txt - The full string to parse out
  2. numLines - The number of lines you want to cut it into
  3. charPerLine - The number of characters you want per line 
$opt={}
$opt.createString =function(txt,numLines,charPerLine){
        if(typeof(numLines)!="number")numLines=2;
        if(typeof(charPerLine)!="number")charPerLine=20;
        var finalText="";
        var mySplit=txt.split(" ");
        var line=[],pos=[],cut=[],text=[];
        for(var i=0;i<numLines;i++){
                line[i]=0;pos[i]=0;cut[i]=0;
                text[i]="";
        }
        var j=0;
        for(var i=0,len=mySplit.length;i<len;i++){		
                line[j]+=mySplit[i].length+1; 
                var offset=(j+1==numLines?2:1);
                if(line[j]>charPerLine-offset){
                        pos[j]=i;
                        cut[j]=charPerLine-line[j]+mySplit[i].length;
                        text[j]+=mySplit[i].substr(0,cut[j]);					
                        if(text[j+1]!=null && line[j+1]!=null){
                                text[j+1]+=mySplit[i].substr(cut[j],mySplit[i].length)+" ";
                                line[j+1]+=text[j+1].length;
                        }
                        j++;
                }
                else text[j]+=mySplit[i]+" ";
                if(!(j<numLines)) break;
        }
        for(var i=0;i<numLines;i++)finalText+=(i?"<br/>":"")+text[i];
        return finalText+"â€¦";
}

var newText=$opt.createString(sName,2,25);
alert(newText);
