Controlling the height of a dynamically-created block node (DIV)

We can set the height of a block node to a specific value, say 75px. Most browsers today will set the div to that height - by setting style {height:75px} for that node - regardless of whether the content "fills" it. But what do you do when you have TOO much content and the height extends beyond your intended height? Then you need some way to detect this condition and remove enough "content" to allow it to fit with the intended height.

More on the Scenario

  1. We have a client that uses IDOL to retrieve products that a visitor may like, based on their browsing history. IDOL returns content populated into 3 areas:
    • Product Name area, whose size varies.
    • Product Description area, whose size varies.
    • Product image, price, and "buy" button area, which has a fixed size.
  2. The first 2 areas are on top of each other. We would like both of them to be 75px high.

How the code works

I have included code for a single DIV whose size needs to be set to 75px, here the second one, the "Product Description". This can be repeated for additional DIVs. The code works as follows:

  1. Each area was given an id of "opAreaB" followed by a number - opAreaB0, opAreaB1, etc - so we use that as the marker for the DIV.
  2. When creating the DIV, you don't give it any height parameter, you simple add the text to create the visible DIV.
  3. Once all the DIVs are created, the code begins.
  4. The code finds the DIVs in question - using the opAreaB marker.
  5. The code detects the height of the DIV.
    • If it is NOT greater than 75px, no need to do anything, just set the height to 75px.
    • If it is greater than 75px, then we:
      • Remove characters from the HTML of the DIV - here I picked 8.
      • Repopulate the DIV with the new HTML, adding a "..." at the end indicating this is not all the description
      • Check to see if it is still greater than 75px. If so, repeat this process, if now, you are finished.

Code

Here is the code. 

//Look for DIVS that include "opAreaB" and make sure the height is under 75px.  If it is not
var divs=document.getElementsByTagName("div");
var MAX_HEIGHT=75;
var REMOVE_CHAR_NUM=8;
for(var i=0;i<divs.length;i++){
        if(divs[i].id&&divs[i].id.indexOf("opAreaB")!=-1){
                var divHeight='';
                if(divs[i].offsetHeight){divHeight=divs[i].offsetHeight;}  
                else if(divs[i].style.pixelHeight){divHeight=divs[i].style.pixelHeight;}
                if(divHeight>MAX_HEIGHT){
                        do{
                                var inner=divs[i].innerHTML;
                                inner=inner.replace("\u2026","");
                                var newInner=inner.substring(0,inner.length-REMOVE_CHAR_NUM);
                                divs[i].innerHTML=newInner+"\u2026";
                                if(divs[i].offsetHeight){divHeight=divs[i].offsetHeight;}  
                                else if(divs[i].style.pixelHeight){divHeight=divs[i].style.pixelHeight;}
                        }while(divHeight>MAX_HEIGHT)
                }
                divs[i].style.height=MAX_HEIGHT+"px";
        }
}
