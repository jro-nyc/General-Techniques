PORT numbers - What if the customer's QA environment uses a port other than 80?

From Optimost Wiki

Jump to: navigation, search

Editing PORT numbers

The port, if it exists, is the number with the semicolon (:) after the hostname. For example, www.optimost.com:120, has the port 120. The default port is 80, but sometimes clients will use a different port for their staging environment. If this is the case, then you will have to set the port for the links within the template.

The example below takes a relative url and makes it absolute, including attaching the port number if it is different from the default.

Known issues: This does not work in Conqueror.

Getting and Setting the PORT

To get the port use .port on the location or a link object. The port must have a ":" in front, so you need to add that to the port number, if there is one. So if the port has any value except for 80 or 0, we will use it. 

var thePort = (loc.port && parseInt(loc.port) != 80 && parseInt(loc.port) != 0) ? loc.port : "";
if(thePort != "" && thePort.toString().indexOf(":") == -1) thePort = ":" + thePort;

To set the port, just append it after the hostname, if the hostname does not already have a port number: 

url = loc.protocol + "//" + loc.hostname + (loc.hostname.indexOf(':')==-1?thePort:'')+ url;

This code accounts for a Safari quirk. For the Location object, location.hostname will not give you the port number, but location.host will give it to you. In Safari, it's the opposite, so here I check to make sure it is not there.

Getting and Setting the PORT Javascript

Here is the code for getting the port number, with pertinent comments. You might want to consider always using this, since it accounts for any port number. 

<base href="http://www.optimost.com:120">

<script>
var url = "/index.html";
//Only make the url absolute if it doesn't begin with http
if (url.toLowerCase().substr(0,4) != "http")
{
	//Use the location object OR the base statement if it exists
	var loc = location;
	var useBase = false;
	if(document.getElementsByTagName('base')[0]){
		var base = document.getElementsByTagName('base')[0];
		loc = document.createElement('a');
		loc.href = base.href;
		useBase = true;
	}
	/*
	Only get the port if it exists and it isn't 80 (default) or 0 (Safari gives this for port 80)
	If we have a port, check if the : exists, if not, add it to the beginning of thePort
	*/
	var thePort = (loc.port && parseInt(loc.port) != 80 && parseInt(loc.port) != 0) ? loc.port : "";
	if(thePort != "" && thePort.toString().indexOf(":") == -1) thePort = ":" + thePort;
	
	/*
	Make the url absolute path.
	Add the port after the hostname
	Check to see if the hostname already includes the port number.
	*/
	if (url.substr(0,1) == "/") 
	{
		url = loc.protocol + "//" + loc.hostname + (loc.hostname.indexOf(':')==-1?thePort:'') + url;
	}
	else 
	{
		var path=loc.pathname.substr(0,loc.pathname.lastIndexOf('/'));
		url = loc.protocol + "//" + loc.hostname +  (loc.hostname.indexOf(':')==-1?thePort:'') + path + 
			((url.indexOf('/')==-1) ? '/' : '') + url;     
	}
	if(useBase)loc=null;
}
alert(url);

//The url should now be http://www.optimost.com:120/index.html
</script>
