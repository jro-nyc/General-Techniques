Introduction

Below are some lovely tools that are recommended for all Technical Consultants to use when debugging and developing templates

Name  Description   
Firebug!The essential Firefox debug tool
ScreengrabTakes a screenshot of an entire webpage (so you don’t have to scroll down, take multiple screenshots, and paste them together)
Submit to TabLets you open the next page in a funnel in a new tab/window by holding the control key while you click the submit button.
ColorzillaLets you select a color from the page and gives you its RGB and hex values.
MeasureItProvides a little ruler tool to measure pixels on a webpage.
Web Developer ToolbarRuler, Edit Css, Forms, Images and Cookies
User Agent SwitcherSwitching user agents
URL ParamsChanging url parameters
Add n Edit CookiesCookie editor
Live HTTP HeadersSniffs outs posts and gets
GreasemonkeyAppend javascript on live pages (and make them stick!)
TabintaAllows you to insert tab characters with the tab button in text areas.
