A/B versus MVT

  1. During exploratory waves, multiple tests are conducted for the different areas of the page. Sample is focused at the value level and the sector variable report is emphasized.
    • Sample is highest at the value level (basing decisions on value performance)
    • Sample is lower at the creative level (not making decisions on creative performance)
  2. During confirmatory testing or A/B testing, sample is focused at the creative or page level. Creative wave summary report is emphasized.
  3. Differences in test types are important particularly when assessing whether or not to end wave or begin making decisions based on data.
  4. If testing multiple variables, MVT is the way to go. If you are testing one variable or a concept test, A/B fashion is preferred.

MVT example page

We will look at a simple example to show how powerful an MVT can be in efficiently using sample to test the variables within an experiment. Here is a 5 variable test, with each variable having 2 values.

In this test, we have 32 possible total creatives. But for this example, we will test 8 of those 32. So let's assume we have 6624 creatives evenly spread across those 8 creatives evenly, or 828 views per creative.

You can see that although each creative has 828 views, each value - since, in this balanced designed, each value is used in 4 of the 8 creatives - is seen 4 times as often, for a total of 3312 views. That means that:

  1. If we are interested in data about the creatives, since we have split the traffic across 8 creatives, we will need 8 times more traffic to come to statistical significance.
  2. If we are interested in data about the variables, since each value is shared in 4 of the 8 creatives, we will need 2 times more traffic to come to statistical significance.

MVT: Sector Variable Report

Let's look at the Sector variable report for a test we ran. There are 4 variables in this test, and we are running 2 values for each variable in this wave. We have 8 creatives in this wave.

As with our example above, you will notice that for every two views, you get one view per value. You will also notice that 2 of the variables - C and D - have significant impact, and A has marginal impact. E is a wash.

MVT: Wave Summary or Creative Level Report

Now let's look at the Creative level report for the test.

Now you notice that every creative has one-eight of the traffic, which means that the value level sample is 4 times greater than creative level. And you can see Creative #109 has the best performance.

MVT: Value Matrix and Variable Support

Looking at the value matrix, you will notice that A=1 and C=6 are in creative #109, which supports this result.

Value Matrix and Verification Wave

Now we look at creating a verification wave. We will run 2 creatives again - #105 and #109 - and now create a new creative that has not been run before, containing all control values except C=6. Now we run this new wave for verification:

  1. To see if the champions continue to do well
  2. To see, based on the variable level information, if a creative with all control and C=6 runs well.
